# Screenshots

This folder contains screenshots demonstrating the I Am Smarticus! quiz application in action.

## Recommended Screenshots to Add:

1. **welcome-screen.png** - User registration screen with email, password, name, and age input
2. **category-selection.png** - Main menu showing the three quiz categories
3. **science-quiz.png** - Example of Science quiz questions
4. **maths-quiz.png** - Example of Maths quiz questions
5. **ca-quiz.png** - Example of Computer Applications quiz questions
6. **correct-answer.png** - Feedback when user answers correctly
7. **wrong-answer.png** - Feedback when user answers incorrectly
8. **score-display.png** - Final score and percentage display
9. **certificate.png** - Certificate of Excellence for scores above 70%
10. **retry-option.png** - Screen showing retry mechanism
11. **age-restriction.png** - Error message for users under 13 years
12. **quit-option.png** - Quit functionality demonstration

## How to Add Screenshots:

1. Run the application
2. Use a screenshot tool (Windows Snipping Tool, Snip & Sketch, or Print Screen)
3. Capture the relevant screens
4. Save the images in this folder with descriptive names
5. Update the main README.md to reference these screenshots

## Screenshot Guidelines:

- Use PNG or JPG format
- Keep file sizes reasonable (under 1MB each)
- Use clear, readable console output
- Capture full terminal window for context
- Name files descriptively (e.g., `science-quiz-example.png`)
